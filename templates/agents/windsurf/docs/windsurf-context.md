# Windsurf — Aura-SDD v{{VERSION}}

This project uses Aura-SDD for Spec-Driven Development. Rules are in `.windsurf/rules/aura/`.

## Aura-SDD Workflow

Use Cascade to invoke:

- `aura-discovery <idea>` — Route new work
- `aura-constitution` — Project principles
- `aura-spec <feature>` — Requirements (EARS)
- `aura-plan <feature>` — Architecture + Boundaries
- `aura-tasks <feature>` — Task decomposition
- `aura-impl <feature>` — Autonomous implementation
- `aura-validate <feature>` — Validation

## Key disciplines

- Boundary-First development
- Constitutional Gates during planning
- EARS format acceptance criteria
- 1-task-per-iteration for context hygiene
