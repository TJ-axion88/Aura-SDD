# AGENTS.md — Aura-SDD v{{VERSION}}

This project uses Aura-SDD for Spec-Driven Development. Skills are in `.codex/skills/aura-*/`.

## Aura-SDD Skills

```
/aura-constitution       → Project principles
/aura-steering           → Project memory
/aura-discovery <idea>   → Route new work (start here)
/aura-spec <feature>     → EARS-format requirements
/aura-clarify <feature>  → Resolve ambiguity
/aura-plan <feature>     → Architecture + Boundary Commitments
/aura-tasks <feature>    → Task decomposition
/aura-impl <feature>     → Autonomous implementation
/aura-validate <feature> → GO/NO-GO validation
/aura-issues <feature>   → GitHub Issues from tasks
```

## Disciplines

- Boundary-First: Identify boundary contracts before implementation
- Constitutional Gates: Check principles before planning
- EARS format: WHEN/IF/WHILE/WHERE/THE SYSTEM SHALL for all acceptance criteria
- Phase gates: Human approval at each phase transition
